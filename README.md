
#  Kame Game

Kame Game esta desarrollado con **React + Vite + TypeScript y Expressjs + Prisma**.  
El proyecto usa **React Router** para la navegación y ESLint para mantener un código limpio y consistente.

## Requisitos Previos
Antes de empezar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) v18 o superior  
- [npm](https://www.npmjs.com/) 

---

##  Instalación

Clona el repositorio y entra al directorio del proyecto
cd kame-game/frontend

Instala las dependencias
npm install

Ejecuta el proyecto
npm run dev

Clona el repositorio y entra al directorio del proyecto
cd kame-game/backend

Instala las dependencias
npm install

Activa la base de datos
npx prisma generate

Ejecuta el proyecto
npm run dev
